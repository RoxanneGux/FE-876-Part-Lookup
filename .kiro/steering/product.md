# Product Overview

This is an Angular "harness" (sandbox) app for ticket **FE-876**. It provides a clean-room prototyping environment that uses the corporate design system (`@assetworks-llc/aw-component-lib`) to build and validate UI features before they are integrated into the main FA Suite application.

The harness is used by UX designers and developers to:
- Prototype new UI features with real component library components
- Validate designs against Figma mockups
- Produce working reference code that engineering can copy into the main app
- Demo specific UI states to stakeholders using mock data

The audience for the running harness is UX reviewers and front-end engineers. The audience for the codebase is the developer who will integrate it into FA Suite.
